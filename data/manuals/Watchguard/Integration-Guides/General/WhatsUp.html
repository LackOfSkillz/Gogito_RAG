WhatsUp Gold Integration Guide

WhatsUp® Gold monitors, reports, alerts, and takes action on the status of network devices, systems, and services. It also installs, discovers, and maps topology and network connected assets. WhatsUp leverages SNMP v1/2/3 and WMI to keep the network running and provides alerting and notification capabilities to keep you informed about issues on your network.

This document describes the steps to integrate your WatchGuard Firebox with WhatsUp Gold.

Platform and Software

The hardware and software used to complete the steps outlined in this document include:

  • WatchGuard Firebox installed with Fireware v12.7 or higher
  • WhatsUp Gold 2021 (21.0.2 Build 175), Total Plus Edition installed on Windows Server 2019 Standard Edition

Set Up the Firebox

Before WhatsUp can discover your Firebox, you must configure SNMP settings in Fireware Web UI.

To configure SNMP settings and add a firewall policy, from Fireware Web UI:

  1. Select System > SNMP.
  2. From the Version drop-down list, select v3.
  3. From the Authentication Protocol drop-down list, select SHA1.
  4. In the Password and Confirm text boxes, type the authentication password.
  5. From the Privacy Protocol drop-down list, select DES.
  6. In the Password and Confirm text boxes, type the encryption password.
  7. In the User Name text box, type a name.
  8. Click Save.

  1. Select Firewall > Firewall Policies.
  2. Click Add Policy.
  3. Create an SNMP packet filter policy that allows SNMP traffic from Any-Trusted to Firebox.

    4. If you connect to an optional interface, specify Any-Optional instead of Any-Trusted.

Set Up WhatsUp

The steps to set up WhatsUp include:

  • Scan the network
  • Turn on monitoring
  • Add an SNMP active monitor for the device
  • Add a performance monitor for disk utilization

To scan the network, from the WhatsUp web UI:

  1. Log in as an administrator.

  2. Select Discover > New Scan.
    If this is the first time you access the site or no device is under monitor, you can also click Start Scan Now in the center of the page.

  3. In the Start step of the Discovery Scan wizard, select the Single device, multiple devices, IP ranges or subnets check box, and type the IP address of your Firebox. Click Next.

  4. Click Next in Expand and Limit page. keep default settings.
  5. In the Credentials step, click to expand the drop-down list.

  6. Select SNMP v3.
    The Add SNMP v3 Credential dialog box opens.
  7. Specify the information shown in this image. Click Save. The password should match the password set in the Firebox.

  8. In the list, select the SNMP v3 credential you created. Click Next.
  9. Click Next again to go to the last step of the wizard.
  10. In the Name text box, type a name for the scan.

  11. Click Save & Run Now.

To turn on monitoring:

  1. Select Discover > Discovered Network.
  2. Select your device in the list.
  3. In the right pane, click Start Monitoring.

  4. In the pop-up that opens, click Start.

To add an SNMP active monitor for the device:

  1. Select the My Network tab and select your device.
  2. In the data sheet, click in the upper-right corner and select Device Properties.

  3. Click and select Active Monitor.

  4. From the drop-down list, select SNMP. Click Next.

  5. Select a network interface. Click Next.
  6. Select the Create Action set for this device only option.
  7. Click Add.
    The Action Builder dialog box opens.
  8. Specify the action properties. Click OK.

  9. Click Finish.

To add a performance monitor for disk utilization:

  1. Click and select Performance Monitor > Use a library monitor.

  2. In the Configure Performance Monitor dialog box, select Disk Utilization.
    Other monitors are available by default.
  3. Click Next.
  4. Select the All disks option.
  5. Click Finish.

Test the Integration

  1. In the WhatsUp Gold web UI, select the MY NETWORK tab.
  2. Select your device.
  3. In the data sheet, click in the upper-right corner and select Device Status.
    The Monitoring page shows basic information about the device.

  4. To show information on CPU and memory utilization, click Disk/CPU/Memory.

  5. To show interface utilization and other interface information, click Router/Switch/Interface.

  6. To show device information, status, and statistics, click General.
